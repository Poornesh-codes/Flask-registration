# Flask Registration App

A simple Flask project demonstrating **user registration** functionality. This project is designed to help beginners understand **Flask routing, templates, and form handling**.

## Features

- User registration form

- Rendering HTML templates with Flask
- Basic project structure for Flask apps


## Installation

1. Clone the repository:

```bash
git clone https://github.com/Poornesh-codes/Flask-registration.git
cd Flask-registration
```
2 Install dependencies
pip install -r requirements.txt

## Usage

Run the Flask app
